# 0Pod

> Zero Limits. All Music.

A universal music player PWA that replicates the iconic Apple iPod Classic experience in a modern browser. Aggregates YouTube Music, Spotify, SoundCloud, and local files into a unified library — all navigated through a tactile click wheel with authentic haptic feedback.

## Status

Phase 0: iPod shell + click wheel + haptics + menu navigation (no audio)

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite
- **Backend:** Node.js + Fastify
- **Monorepo:** Turborepo + pnpm
- **State:** Zustand
- **Styling:** CSS Modules
- **Haptics:** Web Vibration API (Samsung S23 optimized)
